# Wireframes & Maquettes UX

## Outils recommandés

- **Figma** : Design collaboratif, prototypes interactifs
- **Sketch** : Alternative (macOS uniquement)
- **Adobe XD** : Suite Adobe

## Écrans prioritaires (MVP)

### 1. Onboarding
- [ ] Écran d'accueil (splash)
- [ ] Inscription (numéro/email)
- [ ] Vérification OTP
- [ ] Création profil (nom, photo)
- [ ] Permissions (contacts, notifications)

### 2. Liste de conversations
- [ ] Liste threads (avatar, nom, dernier message, timestamp, badge non-lu)
- [ ] Search bar
- [ ] FAB "Nouvelle conversation"
- [ ] Bottom nav (Chats, Appels, Contacts, Paramètres)

### 3. Conversation 1:1
- [ ] Header (avatar, nom, statut en ligne, actions)
- [ ] Liste messages (bulles, horodatage, statuts envoi/lu)
- [ ] Composer (input, emoji, pièce jointe, micro, envoyer)
- [ ] Indicateur "En train d'écrire..."
- [ ] Bandeaux système (clé changée, message éphémère activé)

### 4. Groupe
- [ ] Header groupe (avatar, nom, membres count)
- [ ] Messages avec nom expéditeur
- [ ] Infos groupe (membres, permissions, paramètres)

### 5. Recherche
- [ ] Barre de recherche globale
- [ ] Résultats par catégorie (Messages, Contacts, Groupes)
- [ ] Filtres (date, expéditeur)

### 6. Appels
- [ ] Écran appel entrant (accepter/refuser)
- [ ] Écran appel actif (mute, caméra, speaker, raccrocher)
- [ ] Mini preview vidéo (PIP)

### 7. Paramètres
- [ ] Profil (édition nom, photo, bio)
- [ ] Confidentialité (qui voit quoi, blocage)
- [ ] Notifications
- [ ] Stockage (gestion cache)
- [ ] Sécurité (2FA, devices, export clés)
- [ ] À propos

## Principes de design

### Accessibilité (WCAG 2.1 AA)
- Contraste texte ≥ 4.5:1
- Taille police ≥ 16px (corps de texte)
- Zones tactiles ≥ 44×44 px
- Support lecteurs d'écran
- Navigation clavier (web)

### Responsive
- Mobile-first (320px min width)
- Breakpoints : 320, 768, 1024, 1440 px
- Layout adaptatif (colonnes, grilles)

### Thèmes
- Clair (défaut)
- Sombre (OLED-friendly)
- Auto (selon système)

### Icônes
- Système natif (iOS SF Symbols, Android Material Icons)
- Custom icons en SVG (optimisées)
